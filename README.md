# 船舶智能导航路径规划数据集可视化演示平台

本应用使用 `Streamlit + Folium` 将已有 AIS 清洗平滑、航道拓扑、动态边权和多目标路径规划成果组织为比赛展示界面。

## 功能

- `项目总览`：核心成果指标、数据口径说明与技术链路。
- `数据治理`：清洗流程、卡尔曼滤波降噪指标及报告成果图。
- `航道网络`：按案例区域浏览节点热度、高频边和高置信航段。
- `动态边权`：查看置信度分级、昼夜耗时、路段弹窗与统计图。
- `智能规划演示`：对比三个既有典型案例，或输入起终点运行 A* / Dijkstra 自定义规划。
- 导出路线摘要、路段明细 CSV 与 Folium HTML 地图。

## 安装与运行

在本目录执行：

```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

浏览器打开 Streamlit 提示的本地地址即可进入界面。

如果希望不修改全局 Python 环境，也可将地图组件安装在项目本地目录；应用会自动加载 `.vendor`：

```powershell
python -m pip install --target .vendor folium streamlit-folium
python -m streamlit run app.py
```

## 推荐演示流程

1. 在 `项目总览` 介绍 AIS 数据到导航路线的完整成果链。
2. 在 `数据治理` 展示卡尔曼滤波降噪效果。
3. 在 `航道网络` 与 `动态边权` 点击航道边，解释真实轨迹和置信度如何支撑选路。
4. 在 `智能规划演示` 选择 `西江内河`，同时开启三条路线，突出综合最优路线可靠性提升。
5. 切换 `自定义规划`，设置船舶参数并导出规划摘要或 HTML 地图。

## 数据与边界说明

- 现有交付物包含 `12,305` 个航道节点、`47,826` 条有向边和 `30,106` 条精细化统计边。
- 报告中的权重建模有效事件为 `241,224` 次；交付权重文件直接映射到路网后可汇总的有效通行次数为 `227,903` 次，应用中分别标注两种口径。
- 当前边表中的吃水和限高字段为约束机制演示占位值（默认 `20m / 50m`）。系统可以执行约束过滤，但结果不能替代真实电子海图、桥梁净空或海事通告。
- 本展示层读取既有成果，不重新运行原始 AIS 清洗过程，也不提供实时船舶在线导航。
